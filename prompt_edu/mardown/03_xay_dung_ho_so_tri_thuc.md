# 03. Xây dựng hồ sơ tri thức

## Mục tiêu
Chuyển tập chunk thành bản đồ tri thức của chương học.

## Đầu vào
- Các chunk được truy xuất ở Bước 2

## Công việc thực hiện

### 1. Phân tích cấp chương
- Cần nắm gì
- Cần hiểu gì

### 2. Phân tích bài học
- Kiến thức trọng tâm
- Nội dung cần hiểu

### 3. Phân tích dạng bài
- Các dạng bài
- Độ khó
- Kiến thức yêu cầu

### 4. Xây dựng quan hệ kiến thức
```text
Kiến thức A
   ↓
Kiến thức B
   ↓
Kiến thức C
```

### 5. Xây dựng quan hệ dạng bài
```text
Dạng A
  ↓
Dạng B
  ↓
Dạng C
```

## Đầu ra
```yaml
chương:
cần_nắm:
cần_hiểu:
bài_học:
quan_hệ_kiến_thức:
quan_hệ_dạng_bài:
```
